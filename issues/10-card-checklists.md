# Modal de card — checklists

## What to build

Adicionar suporte a checklists no modal de card. O usuário pode criar múltiplas checklists, cada uma com título e itens marcáveis.

## Acceptance criteria

- [ ] Modal exibe todas as checklists do card com seus itens
- [ ] EDITOR e OWNER podem criar uma nova checklist com título
- [ ] EDITOR e OWNER podem adicionar itens a uma checklist
- [ ] Qualquer membro ACTIVE pode marcar/desmarcar itens (checked/unchecked)
- [ ] EDITOR e OWNER podem deletar um item ou uma checklist inteira
- [ ] Progresso da checklist exibido como barra ou contagem (X/Y itens concluídos)
- [ ] Server Actions validam sessão e role

## Blocked by

- #5 (Cards — criar, editar título/descrição, deletar)
