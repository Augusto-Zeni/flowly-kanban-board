# Modal de card — tags coloridas

## What to build

Adicionar suporte a tags coloridas no modal de card. O usuário pode criar tags com um label e uma cor (seletor de cor com paleta pré-definida), associá-las ao card e removê-las.

## Acceptance criteria

- [ ] Modal de card exibe as tags existentes do card
- [ ] EDITOR e OWNER podem adicionar uma nova tag (label + cor da paleta)
- [ ] EDITOR e OWNER podem remover uma tag do card
- [ ] Tags são exibidas visualmente nos cards na coluna (como chips coloridos)
- [ ] Server Actions validam sessão e role
- [ ] `touchBoard` chamado em adicionar e remover tag

## Blocked by

- #5 (Cards — criar, editar título/descrição, deletar)
